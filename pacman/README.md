# 吃豆人游戏 (Pac-Man)

经典吃豆人游戏的Python实现，使用pygame库开发。

## 项目结构

```
pacman/
├── constants.py    # 游戏常量定义（颜色、尺寸、速度等）
├── maze.py         # 迷宫地图模块
├── pacman.py       # 吃豆人角色模块
├── ghost.py        # 幽灵敌人模块
├── game.py         # 游戏主逻辑和控制模块
├── main.py         # 游戏入口文件
└── README.md       # 项目说明文档
```

## 安装依赖

需要安装pygame库：

```bash
pip install pygame
```

## 运行游戏

```bash
cd pacman
python3 main.py
```

## 操作说明

- **W** - 向上移动
- **A** - 向左移动
- **S** - 向下移动
- **D** - 向右移动
- **ESC** - 暂停/继续游戏
- **SPACE** - 开始游戏/重新开始

## 游戏规则

1. 使用WASD键控制吃豆人在迷宫中移动
2. 吃掉所有的小豆子（白色小圆点）即可获胜
3. 吃掉能量豆（大圆点）可以让幽灵变成可食用状态，此时可以吃掉幽灵获得200分
4. 被幽灵碰到会损失一条命，共有3条命
5. 普通豆子：10分，能量豆：50分，吃掉幽灵：200分

## 幽灵角色

- **红色幽灵(Blinky)**: 直接追击吃豆人
- **粉色幽灵(Pinky)**: 预测吃豆人的移动方向并提前拦截
- **青色幽灵(Inky)**: 从反方向追击吃豆人
- **橙色幽灵(Clyde)**: 部分时间随机移动，部分时间追击吃豆人
